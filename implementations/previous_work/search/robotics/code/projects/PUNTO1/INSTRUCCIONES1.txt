1. Las ecuaciones de cinemática directa se encuentran en el archivo cinematica.m en la carpeta de FUNCIONES. Acá también se encuentran las otras funciones utilizadas para el numeral 2, donde se encuentran las colisiones y se genera el c-space. 

2. En el archivo RUN.m se encuentra el punto b, c, y d. Aca se puede observar el C-space con los obstáculos, utilizando diferentes valores de Z, así como el cspace en 3D. Este archivo utiliza las funciones cspacerSCARA2d y cspaceSCARA3d. También se muestra como se define la trayectoria para el robot SCARA en el archivo trayectoria.m.

3. En el archivo SIMULACION1.slx se observa la simulación con simscape, que también queda demostrada en el video PUNTO1.mp4. Para que corra la simulación es necesario subir la ubicación del STL correspondiente a cada objeto, ya que esta cambia con cada PC.

*Si sale un error es porque debe agregar todas las carpetas al path.
